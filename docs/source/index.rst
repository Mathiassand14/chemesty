Welcome to Chemesty's documentation!
==================================

Chemesty is a Python library for chemistry calculations and molecular modeling. It provides tools for working with chemical elements, molecules, and databases of chemical compounds.

Features
--------

* Comprehensive database of chemical elements with properties
* Molecule creation and manipulation
* Chemical calculations (molecular weight, volume, density, etc.)
* Database storage and retrieval of molecules
* Integration with common chemical file formats
